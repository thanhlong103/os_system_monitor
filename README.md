# OS System Monitor

This is a simple terminal-based system monitor designed to provide an overview of your system's performance. It displays useful information about CPU usage, memory, tasks, IO, and processes.

## Features

- Full-screen view of system information.
- Displays information about CPU usage, memory usage, tasks, IO, and running processes.

## Installation

Follow these steps to clone and compile the project:

1. Clone the repository:
```ruby
   git clone <repo-url>
```

2. Change the directory to proc:
3. 
   cd proc
